The \hyperlink{classApp}{App} class represents the main user interface of the program. It is initialized by the Json\+Input class when an input file is passed. The app holds the \hyperlink{classAction}{Action} class, the main Gluon\+Field object and the \hyperlink{classObservable}{Observable} vector. The generic execute() method is used to start the execution of the main algorithm.
